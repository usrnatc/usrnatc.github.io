About The GameBoy Colour is a handheld video-game console and in my opinion is a really neat piece of tech. First released in 1998, the GBC boasted some really nice hardware, and some not so nice hardware. It amazes me to this day that the console was released with an LCD screen and no backlight; probably the most annoying downfall of the entire console.

See below for a general run-down of the hardware which I'll keep adding to when I find more information.

I'll be honest I didn't own many consoles as a kid, but the GameBoy is probably my favourite out of the ones I did have; super portable, yeah the 2 AA batteries were annoying to find around the house but they seemed to last an eternity, the absolute plethora of games in the console's catalogue, plus, probably the most contradicting take, the console felt bulky and compact at the same time: hard to lose but comfortable to handle.


CPU
  • Sharp LR35902 processor
  • 4 MHz / 8 MHz clock speed
  • 8 bit data bus
  • Essentially a Zilog Z80 processor with the Intel 8080 instruction set (with some more changes)


    • RAM
  • 16 bit addresses
  • 32kB internal memory
  • Up to 32kB external expansion


    • VIDEO
  • Reflective LCD
  • 160*144 dots (physical)
  • 16kB internal memory
  • 40 sprites (of 8*8 or 8*16)


    • COLOUR PALETTE
  • 10, 32, or 56 colours on screen
  • Supports 4 colours per sprite (1 being transparency)


    • ROM
  • 256 bit bootstrap (for use on CPU failure)
  • Total size seems to vary from source to source


    • SOUND
  • Single mono internal speaker
  • Stereo headphone jack port
  • 4 sound channels (each of which can be in L, R, or LR mode
  • Single programmable PCM 4 bit wave channel
  • 64 4 bit samples stored as 1*64 bank array or 2*32 bank array
  • Single noise generator
  • Single audio input from cartridge
  • 2 square wave channels


    • COMMUNICATION
  • Single serial port
  • Infrared communications port


    • POWER
  • 3 Volts over 0.2 Amps yeilding 0.6 Watts
  • 2 AA batteries
